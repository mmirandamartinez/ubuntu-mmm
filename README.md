# Ubuntu MMM — Simulación de Escritorio Web

Experiencia de escritorio totalmente interactiva construida íntegramente con **HTML, CSS y JavaScript puro**. Sin frameworks ni dependencias externas.

> Demo en vivo: [mmirandamartinez.com](https://mmirandamartinez.com/login)

---

## Características

El proyecto simula un entorno de escritorio con aplicaciones funcionales:

- Firefox: navegador web simulado.
- Administrador de archivos: explorar carpetas y ficheros.
- Terminal: emulador interactivo de línea de comandos.
- Tareas: app de lista de tareas.
- Tienda de apps: instalar/desinstalar apps (por ejemplo Tetris, Snake).
- Calculadora: calculadora operativa.
- Calendario: calendario mensual con navegación.
- Snake: juego clásico de la serpiente.
- Tetris: juego clásico de Tetris (instalable desde la tienda).
- Discord: interfaz de chat tipo Discord.
- Spotify: reproductor de música simulado.
- Monitor de sistema: simulación de uso de CPU/RAM.
- Ajustes: configuración de tema y aspecto.
- Editor de texto: editor de código con Monaco.
- Pantalla de inicio de sesión: bloqueo e inicio de sesión personalizado.

---

## Inicio rápido

No requiere instalación. Abre el archivo en tu navegador:

```bash
git clone https://github.com/mmirandamartinez/ubuntu-mmm.git
cd ubuntu-mmm
open login.html
```

O arrastra `login.html` al navegador.

---

## Tecnologías utilizadas

- HTML5 — Estructura y maquetación.
- CSS3 — Estilos y animaciones.
- JavaScript (vanilla) — Lógica e interactividad.
- Monaco Editor — Componente de editor de código.
- Google Fonts — fuente Press Start 2P.
